GIS Tools

These GIS tools allow the interpolation of environmental data for a specified sampling site or location.

World Ocean Atlas (WOA05) Extractor

World Ocean Atlas 2005 is a set of objectively analyzed (1° grid) climatological fields of in situ temperature, salinity, dissolved oxygen, Apparent Oxygen Utilization (AOU), percent oxygen saturation, phosphate, silicate, and nitrate at standard depth levels for annual, seasonal, and monthly compositing periods for the World Ocean.
Use the WOA05 Extractor

World Ocean Database (WOD05) Extractor

WOD05 is a collection of scientifically quality-controlled ocean profile (time series) that includes measurements of temperature, salinity, oxygen, phosphate, nitrate, nitrite, silicate, chlorophyll, alkalinity, pH and plankton.
Use the WOD05 Extractor
